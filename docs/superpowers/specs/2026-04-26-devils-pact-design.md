# Devil's Pact - 魔鬼契约 设计文档

## 概述
一个哥特风格的"魔鬼交易"网页应用。用户输入愿望，AI返回愿望实现但代价惨重的故事。

## 技术栈
- React + Vite
- 纯CSS动画（哥特风）
- DeepSeek API（前端直调）
- Google Fonts哥特字体

## 页面结构（单页）
1. **背景层**：深红/暗紫渐变 + 烟雾粒子
2. **标题区**：哥特字体 "DEVIL'S PACT" / "魔鬼契约"，烛光效果
3. **输入区**：羊皮纸质感的输入框，古旧花纹边框
4. **提交按钮**：封印图案，hover发光，点击燃烧动画
5. **结果区**：逐字打印故事，代价文字变红加粗
6. **底部装饰**：拉丁文铭文

## 动画
- 加载：雾气涌入，标题浮现
- 等待：五芒星法阵旋转
- 结果：逐字打印+代价高亮
- 背景：烛光摇曳+偶尔闪电

## Prompt设计
魔鬼角色，华丽修辞，每个愿望有致命反噬，风格参考《猴爪》和《浮士德》。
